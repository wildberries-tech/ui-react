## Design Tokens
Предоставляет набор дизайн токенов в виде CSS переменных. Все предоставляемые переменные ограничены областью использования компонента или хука.

Содержит параметры: цветовая палитра, типографика, скругления, тени.

## Использование
Импорт компонента в код.
```tsx
import { DesignTokens } from '@wb-tech/ui-react';
```

Передайте в компонент область в которой необходимо получить доступ к переменным. Ниже приведен наиболее частый случай применения где областью является все приложение.
```tsx
import React from 'react';
import { render } from 'react-dom';

import { DesignTokens } from '@wb-tech/ui-react';

render((
    <DesignTokens>
        <YourAppCode />
    </DesignTokens>
), document.getElementById('root'));
```
